Alt - Team 488 Vision and Localization System
=============================================

Alt is a vision and localization system designed for robotics applications,
particularly for the First Robotics Competition. It provides features for:

* Object detection and tracking
* Reef (field element) localization
* Path planning
* Agent-based architecture for extensibility
